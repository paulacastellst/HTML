Label

L'element HTML té una etiqueta label que serveix per escriure el nom del camp a emplenar. Té etiqueta d'obertura i tencament.Va dintre del form

Input

Button

L'element HTML button crea un botó que permet enviar el formulari.Té etqiueta button d'obertura i tencament.

Aqui us deixo el codi per muntar el primer formulari.
