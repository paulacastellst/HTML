Més Elements

Etiqueta Fieldset

S'utilitza per agrupar elements dins d'un formulari. Es fa servir l'etiqueta fiedset i també l'etqiueta de legend que permet posar un nom al contingut del fiedset. Tñe etiqueta d'inici i tencament. 

        
            <form>
             <fieldset>
                <legend>Ddes personals</legend>
                <label for="name">Nom</label>
                <input type="text" id="name">
                <label for="surname">Cognom</label>
                <input type="text" id="surname">
             </fieldset>
            </form>

Input tipus file

Serveix per afegir un fitxer i per enviar-ho.

        
            <form>
             <fieldset>
                <legend>Ddes personals</legend>
                <label for="name">Nom</label>
                <input type="text" id="name">
                <label for="surname">Cognom</label>
                <input type="text" id="surname">
                <input file="text" name="photo" id="photo">
             </fieldset>
            </form>

Etiqueta Meter

A part de l'atribut indispensable value, porta atribut min, max, low, high i optimum

    
        <form>
            <label for="fuel">Combustible</label>
            <meter id="fuel" min="0" max="100" value="90" low="25" high="75" optimum="100"></meter>
           </form>

Value es el valor de una determinada cosa, low es que esta molt baix, high mitjà i otimum es el màxim

Etiqueta Progress

Té etiqueta d'obertura i tencament. Crea una barra de progressió en funció dels atributs value i max.

        <form>
            <label for="task">Tasca 1</label>
            <progress id="task" max="100" value="60"></progress>
        </form>

Una bona pràctica, és treballar amb percentatges. Dues barres, una ambb max=200 i l'atre de max=100 tenen mateixa longitud. (Tasca 1 max=100)(Tasca 2 max=200)

Aquest value es cambia segons accions, id hauria cambiar task 1

Etiqueta Textarea

Porta atributs cols i rows amb nombres numerics

        <form>
            <div>
                <label for="message">Escriu el teu missatge</label>
            </div>
                 <textarea name="" id="message" cols="80" rows="20"></textarea>
           </form>
