Inputs tipus text

Si fiquem type sense text surt text sol.

Atributs habituals

es fa servir els atributs següents

name

Per saber quin formulari controla cada tipus de dada introduïda

max-length

Serveix per limitar caracters dels camps, porta valor numeric, per triar els valors que es poden ficat

size

Per utilitzar formularis antics per indicar l'amplada de la caixeta

value

s'utilitza per posar un valor determinat

placeholder

Posar un text temporalment, provisional

pattern

Acompanyat amb una expressió regular, inpedeix posar qualsevol text que no siguiel que hem posat

required

Fa que la casella s'hagi d'omplir obligatoriament, elegantment es posa unesterisc.
