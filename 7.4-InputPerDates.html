Input per les dates

Ens deixa introduir dia, mes, any i setmana. Amb tipus date ens deixa en tots els navegadors

Codi

<form>
        <input type="date">
    </form>

Resultat

Input amb tipus datetime

Dexia enviar hores. Està obsolet, es comporta com un camp de tipus text i no s'ha de fer servir.

Codi

<form>
        <input type="datetime">
    </form>

Resultat

Input tipus datetime-local

Dexia posar dia, mes, any i l'hora. No l'utilitzem.

Codi

    <form>
        <input type="datetime-local">
    </form>

Resultat

Input de tipus date i time

Funciona amb tots els navegadors. Dexia posar l'Hora, a més de la data

Codi

    <form>
        <input type="date">
        <input type="time">
    </form>

Resultat

Input tipus month

Deixa posar el mes, també demana l'any

Codi

    <form>
        <input type="month">
    </form>

Resultat

Input tipus week

Demanar setmana en concret, funciona en tots els navegadors. No funciona en motzilla

Codi

    <form>
        <input type="week">
    </form>
